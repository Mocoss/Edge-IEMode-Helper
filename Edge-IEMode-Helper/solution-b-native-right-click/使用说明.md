# 原生右键 IE 模式使用说明

## 一键开启步骤
1. 双击「启用IE模式右键菜单.reg」，点击确认导入注册表
2. 完全关闭 Edge 浏览器（任务管理器结束所有 Edge 进程）
3. 重新打开 Edge 即可使用

## 使用方法
### 方式1：右键菜单切换
在任意网页空白处点击右键 → 选择「在 Internet Explorer 模式下重新加载」，立即切换 IE 内核。

### 方式2：地址栏按钮切换
1. 打开 Edge 设置 → 外观
2. 找到「在 Internet Explorer 模式下重新加载按钮」，开启后固定到地址栏
3. 点击地址栏的 IE 图标即可一键切换

### 方式3：永久添加站点
1. Edge 设置 → 默认浏览器 → Internet Explorer 模式页面
2. 点击「添加」，输入网址后永久生效

## 验证是否生效
切换后地址栏左侧出现小 IE 图标，即为 IE 模式生效。

## 常见问题
### 1. 右键菜单不显示？
导入注册表后必须完全重启 Edge 浏览器，后台进程也要全部结束。
### 2. 有效期多久？
- 右键手动切换：单次生效，关闭标签页后失效
- 设置页手动添加的站点：默认 30 天有效期，可自行设置最长 90 天
- 企业模式站点列表（方案A）：永久有效
### 3. 安全性如何？
这是微软官方内置功能，由微软维护 IE 内核，安全性和兼容性远高于第三方 IE Tab 插件。
